# 📝 Smart Notes App

A modern and responsive **Smart Notes application** built using **HTML, CSS, and JavaScript**, enhanced with **Capacitor** to run as a mobile app.

---

## 🚀 Features

* ✍️ Add and manage notes easily
* 🎤 Voice input (Speech-to-Text)
* 🌙 Dark / ☀️ Light mode toggle
* 🔍 Search notes
* 📱 Mobile-ready using Capacitor
* 💾 Local storage support

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Capacitor (for Android app)

---

## 📂 Project Structure

```
SmartNotesApp/
│
├── www/
│   ├── index.html
│   ├── style.css
│   ├── script.js
│
├── android/
├── capacitor.config.json
├── package.json
```

---

## ⚙️ Installation & Setup

```bash
git clone https://github.com/YOUR_USERNAME/smart-notes-app.git
cd smart-notes-app
npm install
npx cap sync
npx cap open android
```
